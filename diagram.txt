LG IR Remote — Wiring Diagram
==============================

Power rail
──────────
  Li-ion (+) ─── Arduino VCC ─── IR LED anode
  Li-ion (−) ─── Arduino GND ─── 2N2222 emitter (via 50Ω) ─── button returns
                                         │
                                  common ground rail


IR LED driver (NPN switching)
─────────────────────────────

                 VCC (3.7 V Li-ion)
                   │
                IR LED (anode)
                   ▼
                IR LED (cathode)
                   │
               Collector
                   │
  D3 ── 1kΩ ── Base ─┤ 2N2222 (NPN)
                   │
               Emitter
                   │
                  50Ω
                   │
                  GND   (common rail)


Buttons (normally open, internal pull-ups, active LOW)
──────────────────────────────────────────────────────
  D4 ─── pin1 ┤ POWER  ├ pin2 ─── GND
  D5 ─── pin1 ┤ HDMI-1 ├ pin2 ─── GND
  D6 ─── pin1 ┤ HDMI-2 ├ pin2 ─── GND
  D7 ─── pin1 ┤ HDMI-3 ├ pin2 ─── GND

  (buttons are two-terminal momentary switches; either pin
   can go to the MCU pin and the other to GND)


Pin summary
───────────
  VCC  <-  3.7 V Li-ion positive (regulator removed)
  GND  <-  common ground rail: battery (−), Arduino GND,
           2N2222 emitter (via 50Ω), all 4 button returns
  D3   ->  1kΩ -> 2N2222 base (IR LED driver)
  D4   ->  POWER button  -> GND
  D5   ->  HDMI-1 button -> GND
  D6   ->  HDMI-2 button -> GND
  D7   ->  HDMI-3 button -> GND
