LG IR Remote — Wiring Diagram
==============================

                        Arduino Pro Mini 3V3
                       ┌────────────────────┐
                       │                    │
  3.7V Li-ion (+) ────>│ VCC            D3  │──── 1kΩ ──── Base (2N2222)
                       │                    │                  |
  3.7V Li-ion (-) ────>│ GND               │           Collector
                       │                    │                  |
                       │              D4    │───┐        IR LED cathode
                       │              D5    │──┐│             |
                       │              D6    │─┐││        IR LED anode
                       │              D7    │┐│││             |
                       │                    ││││            VCC
                       └────────────────────┘││││
                                             ││││       Emitter (2N2222)
                        Button wiring:       ││││             |
                        (all active LOW,     ││││           100Ω
                         internal pull-ups)  ││││             |
                                             ││││            GND
                        D7 ──┤ HDMI-3 ├──GND ┘│││
                        D6 ──┤ HDMI-2 ├──GND ─┘││
                        D5 ──┤ HDMI-1 ├──GND ──┘│
                        D4 ──┤ POWER  ├──GND ───┘


  IR LED driver detail:
  ─────────────────────

                VCC (3.7V Li-ion)
                 |
               IR LED (940 nm, anode up)
                 |
            Collector
                 |
  D3 ──── 1kΩ ──┤ 2N2222 (NPN)
                 |
            Emitter
                 |
               100Ω
                 |
                GND


  Pin summary:
  ────────────
  VCC  <- 3.7V Li-ion positive (regulator removed)
  GND  <- 3.7V Li-ion negative
  D3   -> 1kΩ -> 2N2222 base (IR LED driver)
  D4   -> Power toggle button  -> GND
  D5   -> HDMI-1 button        -> GND
  D6   -> HDMI-2 button        -> GND
  D7   -> HDMI-3 button        -> GND
